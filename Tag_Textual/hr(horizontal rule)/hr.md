Tag hr berfungsi mewakili jeda tematik anatara elemen tingkat paragraf, contohnya :
- Perubahan adegan dalam cerita, atau pergerseran topik dalam bagian.

BUKAN UNTUK MENAMBAHKAN GARIS HORIZONTAL

Atribut yang bisa digunakan :
- align :
Menetapkan perataan aturan pada halaman. Jika tidak ada nilai yang ditentukan, nilai default nya adalah "left".
- color :
Menetapkan warna aturan melalui nama warna (red,blue) atau hexsadesimal.
- noshade :
Menetapkan aturan untuk tidak memiliki bayangan
- size :
Menetapan tinggi aturan dalam piksel
- width :
Menetapkan panjang aturan pada halaman melalui nilai piksel (5px) atau persentase (8%).

Dampak adanya :
- Menandai perubahan konteks yang memisahkan dua bagian yang tidak satu ide
contoh : artikel -> penulis, konten utama -> komentar, dan satu subtopi -> subtopik lain.

- Screen reader membaca hr sebagai "Separator" / "Thematic break", yang membuat user tahu bahwa sudah ganti topik.

- Struktur dokumen memiliki struktur yang lebih jelas.

Dampak tidak adanya :
- Konten terlihat nyambung padahal beda konteks
- Screen reader tidak diberi sinyal tentang perpindahan topik dan membaca nya seperti satu konteks
